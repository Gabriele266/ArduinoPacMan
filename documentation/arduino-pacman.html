Introduction

Arduino PacMan is a simple tool that simplifies the activities of sharing Arduino projects.
Once you have done a project, with some sources and using some libraries, it will create for you a simple
package, with the sources and the libraries used. It will also generate an installation script that will allow
you to simply install the libraries in the local preferred path.
The only thing that you'll have to do is to run the script.
It will also take care about unpacking the sources into a directory. After this you'll have only to compile the code and... Upload!!

Code informations

Arduino PacMan is written in c++ using the Qt Framework, under the open source licence. The build system used is qmake with gcc in Ubuntu 20.04.
The Qt version is 5.15.2 and the language standard is c++ 14.
The code is organized into some directories, according to the type of their contents.
  • gui Contains all the files related to Graphical widgets or Windows. There are all the forms except the main window, that is away from every folder.
  • packages Contains all the constructs related to the managing of the packages. E.g. the Package class, that represents a package.
  • settings Contains all the constructs related to the managing of the settings for ArduinoPacMan.
  • themes Contains all the constructs used for the managing of the color themes and the themes files.
  • threads Contains all the classes that represent the different thread launched and used by ArduinoPacMan.
  • utils Contains a bunch of functions that are useful for the managing of the constructs.
  • icons Contains all the icons of the project divided by type.
  • documentation Contains this documentation.

Start point

The entry point of the application is the main.cpp file located in the home project directory.
It cares of launching the main window, checking the themes and loading application informations.
To learn more of the theme managing system check the file themes-managing

The main window

The main window is first window that is shown at the startup of the application. It is composed of 3 files:
  • mainwindow.h Contains the header of the window, with all the declarations of methods and
  • mainwindow.cpp Contains the implementation of the the functions.
  • mainwindow.ui Contains the mainwindow graphical design.
