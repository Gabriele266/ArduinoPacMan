Gesione delle impostazioni

Le impostazioni sono organizzate in chiavi, ognuna delle quali ha un nome, un valore e degli attributi o modificatori.
In ArduinoPacMan le impostazioni vengono gestite dalle classi:
  • Settings: per la gestione di un file di impostazioni
  • Group: per la gestione di gruppi di chiavi
  • Key: per la gestione di ogni singola chiave
  • Attribute: per la gestione di un attributo per una chiave
  • AttributeList: per la gestione di una lista di attributi
